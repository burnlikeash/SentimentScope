About SentimentScope

SentimentScope is an advanced product review analysis platform that leverages machine learning to provide comprehensive sentiment analysis and topic modeling for smartphone reviews.

Key Features

  • Sentiment Analysis: AI-powered classification of review sentiment (positive, negative, neutral)
  • Topic Modeling: Automatic identification of key discussion topics in reviews
  • Confidence Scoring: Reviews are prioritized by ML model confidence scores
  • Real-time Data: Live connection to smartphone review database
  • Advanced Filtering: Filter by brand, sentiment, rating, and topics

Technology Stack

  • Frontend: Vanilla JavaScript, HTML5, CSS3
  • Backend: FastAPI, Python
  • Database: MySQL
  • ML Models: DistilBERT, BERTopic, SentenceTransformers

How It Works

Our system processes smartphone reviews through a sophisticated ML pipeline. Reviews are analyzed for sentiment using DistilBERT, and topics are extracted using BERTopic with sentence transformers. High-confidence predictions are prioritized to ensure the most reliable insights are displayed first.
